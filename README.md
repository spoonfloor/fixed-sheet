# Fixed Sheet

A minimal static web app scaffold.

## Preview locally

Open `index.html` directly in a browser, or serve this folder with any static server:

```sh
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Project structure

- `index.html` contains the page markup.
- `styles.css` contains the visual design.
- `script.js` contains small browser interactions.
